# WiFi测试工具

这是一个用于WiFi密码测试的Python工具，具有以下功能：

- 扫描并显示附近的WiFi热点（按信号强度排序）
- 选择WiFi热点和密码字典进行测试
- 多线程测试避免界面冻结
- 显示测试进度和结果
- 支持取消测试操作
- 成功连接后将密码保存到带时间戳的文件中

## 使用方法

1. 运行程序：`python wifitest.py`
2. 点击"扫描"按钮获取附近WiFi列表
3. 从列表中选择要测试的WiFi
4. 点击"选择字典"按钮选择密码字典文件
5. 点击"开始测试"按钮开始测试
6. 测试成功后，密码将保存到带时间戳的文件中

## 依赖

- Python 3.x
- tkinter
- pywifi
